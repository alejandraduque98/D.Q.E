Hola, soy Alejandra Duque diseñadora UI/UX Crea la apariencia (UI) y experiencia de uso (UX) de sitios o apps para que sean atractivos y fáciles de usar. con base en ingeniería de sistemas y experiencia como desarrolladora front-end. Programa lo que ves e interactúas en una web: botones, menús, animaciones, ... Mi enfoque une lo técnico con lo visual para crear experiencias digitales impactantes, centradas en marcas con identidad fuerte. Acompañó a negocios que están dando sus primeros pasos o renovando su presencia digital, y también colaboró con equipos de producto que necesitan escalar sus interfaces Pantallas o elementos visuales con los que una persona interactúa en una web o app. de forma ordenada mediante sistemas de diseño sólidos. Ofrezco soluciones completas: investigación UX Estudio de usuarios reales para entender sus necesidades y mejorar su experiencia con un producto digital. , wireframes Esquemas simples que muestran cómo se organizará el contenido en una pantalla. , prototipos Simulaciones interactivas de una web o app para probar su funcionamiento antes de desarrollarla. , diseño visual Trabajo con colores, tipografías, imágenes y estilo para que una interfaz se vea atractiva y coherente. , Sistemas de diseño Guías visuales y reglas que aseguran coherencia en todo un producto digital (colores, botones, tipografías, etc.). , composiciones fotográficas imágenes cuidadosamente diseñadas para reflejar el estilo y personalidad visual de una marca. para identidad de marca e implementación con HTML Lenguaje que estructura el contenido de una página web (como los textos, títulos e imágenes). , CSS Lenguaje que da estilo a la web: colores, fuentes, tamaños, márgenes... , JavaScript Lenguaje que hace que una web sea interactiva: animaciones, formularios, botones... , REACT Librería de JavaScript que ayuda a construir interfaces de usuario modernas de forma eficiente. , Bootstrap Herramienta con estilos y componentes listos para crear webs responsivas de forma más rápida. .

Reserva Fácil: Diseño Responsive al Estilo Airbnb

Este proyecto es una reinterpretación visual y funcional de una plataforma de reservas, desarrollado en Figma con enfoque en diseño responsive. A través del uso de Auto Layout, variables y componentes reutilizables, se optimizó la experiencia para distintos dispositivos (desktop, tablet y mobile). PlayList CRUD – Tu música, tus reglas

Este proyecto nació como un ejercicio para combinar diseño amigable con lógica funcional, dando vida a una lista de reproducción interactiva donde cada usuario puede gestionar sus canciones favoritas. Fue desarrollado con React y conectado a Firebase, lo que permite almacenar en tiempo real cada nueva canción, edición o eliminación, de forma segura y fluida. La idea detrás de esta app fue crear una experiencia intuitiva donde el usuario no solo interactúe con una interfaz visualmente clara, sino que también sienta control total sobre su contenido. Desde el principio me propuse construir algo más que un formulario: quería que cada parte del flujo fuera coherente, fácil de usar y estéticamente ordenada.
