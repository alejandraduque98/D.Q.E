Acompañó a negocios que están dando sus primeros pasos o renovando su presencia digital, y también colaboró con equipos de producto que necesitan escalar sus interfaces Pantallas o elementos visuales con los que una persona interactúa en una web o app. de forma ordenada mediante sistemas de diseño sólidos. Ofrezco soluciones completas: investigación UX Estudio de usuarios reales para entender sus necesidades y mejorar su experiencia con un producto digital. , wireframes Esquemas simples que muestran cómo se organizará el contenido en una pantalla. , prototipos Simulaciones interactivas de una web o app para probar su funcionamiento antes de desarrollarla. , diseño visual Trabajo con colores, tipografías, imágenes y estilo para que una interfaz se vea atractiva y coherente. , Sistemas de diseño Guías visuales y reglas que aseguran coherencia en todo un producto digital (colores, botones, tipografías, etc.). , composiciones fotográficas imágenes cuidadosamente diseñadas para reflejar el estilo y personalidad visual de una marca. para identidad de marca e implementación con HTML Lenguaje que estructura el contenido de una página web (como los textos, títulos e imágenes). , CSS Lenguaje que da estilo a la web: colores, fuentes, tamaños, márgenes... , JavaScript Lenguaje que hace que una web sea interactiva: animaciones, formularios, botones... , REACT Librería de JavaScript que ayuda a construir interfaces de usuario modernas de forma eficiente. , Bootstrap Herramienta con estilos y componentes listos para crear webs responsivas de forma más rápida. .
